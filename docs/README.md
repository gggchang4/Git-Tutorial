# docs 目录导航 🗂️

本页用于统一浏览 `docs/` 目录下的四本手册，帮助你快速判断“现在应该看哪一份文档”。

## 手册一览 📚

| 文档 | 定位 | 适合什么时候看 |
|------|------|----------------|
| [Git学习手册](./Git学习手册.md) | 从入门到进阶建立 Git 完整认知 | 需要系统学习 Git 概念、流程和原理时 |
| [Git Reference手册](./Git%20Reference手册.md) | 面向日常速查的命令与参数参考 | 知道要做什么，但忘了命令怎么写时 |
| [Git个人使用规范](./Git个人使用规范.md) | 面向个人与小团队的 Git 行为规范 | 想把分支、提交、发布和协作习惯稳定下来时 |
| [Git实用拓展手册](./Git实用拓展手册.md) | 面向工具链、效率技巧与自动化实践 | 想补强 GUI、CLI、钩子、安全防护与效率工具时 |

## 按主题跳转 🔀

- 想系统补 Git 概念：从 [Git学习手册 / Git 简介与安装配置](./Git学习手册.md#1-git-简介与安装配置) 开始
- 想直接查命令：去 [Git Reference手册 / 目录](./Git%20Reference手册.md#目录)
- 想整理分支与提交习惯：看 [Git个人使用规范 / 分支管理规范](./Git个人使用规范.md#分支管理规范) 和 [Git个人使用规范 / 提交规范](./Git个人使用规范.md#提交规范)
- 想补工具链和自动化：看 [Git实用拓展手册 / 自动化工具与脚本](./Git实用拓展手册.md#3-自动化工具与脚本)
- 遇到回滚、恢复或误操作：优先联动 [Git学习手册 / 回滚、撤销与恢复](./Git学习手册.md#5-回滚撤销与恢复) 与 [Git Reference手册 / 回滚与恢复命令](./Git%20Reference手册.md#4-回滚与恢复命令)
- 遇到协作、冲突或 PR 场景：优先联动 [Git学习手册 / 分支管理与协作](./Git学习手册.md#4-分支管理与协作)、[Git个人使用规范 / 小型协作规范](./Git个人使用规范.md#小型协作规范)、[Git实用拓展手册 / 命令行增强工具](./Git实用拓展手册.md#2-命令行增强工具)
