# Git实用拓展手册

## 文档定位

本手册聚焦 Git 的工具链、效率技巧和自动化实践，帮助读者从“会用 Git”进一步走向“用得顺手、用得稳定、用得省时间”。

## 目标内容

- 梳理适合个人开发者的 Git GUI 与命令行工具
- 补充 Git Hooks、Git LFS、敏感信息检测等辅助实践
- 汇总高频操作技巧和常见问题处理技巧

## 章节规划

### 1. GUI 工具

- GitHub Desktop
- Sourcetree
- GitKraken
- VS Code 内置 Git

### 2. 命令行增强工具

- Git alias
- git-extras
- 终端增强与 Git 集成

### 3. 自动化工具与脚本

- Git Hooks
- 提交信息校验
- 提交前格式校验
- 推送前检查

### 4. 大文件与敏感信息防护

- Git LFS
- git-secrets
- 环境变量与安全边界

### 5. 高频技巧

- 快速恢复修改
- 查看历史
- 删除已合并分支
- 远程分支同步
- `cherry-pick`

### 6. 扩展资源

- 官方文档
- 学习网站
- 社区与博客

## 写作要求

- 每个工具尽量说明“用途、适合对象、安装方式、使用场景、优缺点”
- 涉及命令示例时注明适用环境
- 涉及时效性较强的工具状态时，应在写入前先核验
