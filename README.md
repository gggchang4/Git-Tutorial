# 个人 Git 学习与使用参考手册

一个面向个人开发者的 Git 系统化参考项目，目标是把 Git 的学习、查询、规范和效率工具整合成一套长期可维护的 Markdown 手册。

## 项目简介

这个项目基于现有的开发方案与 Codex 开发规范启动，后续将逐步整理为一套适合个人学习、日常查阅和规范实践的 Git 参考资料库。

项目聚焦以下几个方向：

- Git 基础与进阶学习
- 常用命令与参数速查
- 个人 Git 使用规范沉淀
- Git 工具链与效率提升方案整理

## 项目目标

- 做成一套可持续迭代的个人 Git 知识库
- 兼顾系统化学习与快速查询
- 将实践习惯逐步对齐到公开可查的行业规范
- 为后续文档拆分、素材沉淀和版本迭代建立清晰结构

## 规划内容

项目计划拆分为四个核心部分：

1. `Git学习手册.md`
2. `Git Reference手册.md`
3. `Git个人使用规范.md`
4. `Git实用拓展手册.md`

对应内容方向如下：

- Part 1：从入门到进阶的 Git 学习路径
- Part 2：按命令分类整理的 Git Reference
- Part 3：个人 Git 分支、提交与协作规范
- Part 4：Git GUI、命令行工具、Hooks、LFS 与实用技巧

## 当前参考文件

## assets 目录说明

`assets/` 用于统一存放项目开发过程中的素材资源，避免截图、流程图、草稿文件散落在根目录。

- `assets/images/`：插图、封面图、配图素材
- `assets/screenshots/`：命令执行截图、界面截图
- `assets/diagrams/`：流程图源文件、结构图、示意图
- `assets/references/`：临时引用素材、外部整理资料、待筛选资源
